# Task 5: API Integration and Front-End Interaction

## Steps to Run
1. **Navigate to the project directory:**
    ```bash
    cd "Task 5"
    ```
2. **Install the Node Modules:**
    ```bash
    npm install
    ```
3. **Run the Project:**
    ```bash
    npm start
    ```
4. Open your web browser and go to [`http://localhost:3000`](http://localhost:3000)
